# AI Super Agent

A powerful multi-agent research assistant powered by OpenAI, LangChain, and FastAPI — with optional web UI and multi-agent task orchestration via CrewAI.

---

## 🛠️ Prerequisites

- Python 3.10+
- Docker + Docker Compose (for containerized setup)
- OpenAI API Key
- Serper API Key (for web search functionality)
